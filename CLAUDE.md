# Instruções Permanentes — FONTEC

## Idioma
- Responder **sempre em português brasileiro** em todas as interações.

## Fluxo obrigatório de alterações

### Antes de qualquer alteração
1. Fazer **commit e push** do estado atual no GitHub como backup.

### Após concluir as alterações
1. **Validar o HTML** — verificar estrutura, tags emparelhadas e integridade do CSS.
2. Fazer **commit e push** das alterações no GitHub.

## Resumo do fluxo
```
backup commit + push → alterações → validar HTML → commit + push final
```
